# QOR LANG — Quantified Ontological Reasoning Language

Today, building an AI reasoning system requires 4 different tools, 4 different syntaxes, and 4 different mental models — MeTTa for knowledge representation, PeTTa/Prolog for execution, PLN for probabilistic reasoning, and MORK for hypergraph storage.
QORlang collapses all of this into a single unified language. A developer writes .QOR files. The runtime handles everything underneath — pattern matching, probabilistic inference, hypergraph queries, and even enforcing that LLMs output valid QOR syntax.
QORlang is not a replacement for MeTTa, PLN, MORK, PeTTa, or llguidance. It is the unified surface that makes all of them accessible through one coherent language.
